Fluid launches recruiting campaign for a Project Manager

Posted by Fluid Project on 2007-08-06

The Fluid Project is actively searching for a Project Manager.

Detailed information about the position and how to apply is posted on the University of Toronto careers website. The deadline for applications is August 21.

UPDATE: Applications will no longer be accepted as the deadline for submission has passed.

Fluid is a project of the Inclusive Design Research Centre at OCAD University, funded by a grant from The Andrew W. Mellon Foundation.
